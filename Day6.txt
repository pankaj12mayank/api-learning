# 🔥 ✅ **Alternative Project Structure (Service-Based Architecture)**

```text
api-learning/
│
├── main.py
├── models/
│   └── product_model.py
├── routes/
│   └── product_routes.py
├── services/
│   └── product_service.py
├── database/
│   └── db.py
```

---

# 🧠 Why This Is Better?

👉 We separate:

```text
Routes → API layer  
Services → Business logic  
Models → Data structure  
DB → Storage
```

👉 This is how **real backend systems scale**

# 🏗️ Step-by-Step Example

---

## ✅ 1. models/product_model.py

```python
from pydantic import BaseModel

class Product(BaseModel):
    id: int
    name: str
    price: int
```

---

## ✅ 2. database/db.py

```python
product_DB = []
```

---

## ✅ 3. services/product_service.py (🔥 NEW LAYER)

👉 All logic goes here

```python
from database.db import product_DB

def create_product_service(product):
    product_DB.append(product)
    return product

def get_all_products_service():
    return product_DB

def get_product_service(product_id):
    for p in product_DB:
        if p.id == product_id:
            return p
    return None

def update_product_service(product_id, updated_product):
    for index, p in enumerate(product_DB):
        if p.id == product_id:
            product_DB[index] = updated_product
            return updated_product
    return None

def delete_product_service(product_id):
    for index, p in enumerate(product_DB):
        if p.id == product_id:
            product_DB.pop(index)
            return True
    return False
```

---

## ✅ 4. routes/product_routes.py

```python
from fastapi import APIRouter
from models.product_model import Product
from services.product_service import *

router = APIRouter()

@router.post("/products")
def create_product(product: Product):
    return create_product_service(product)

@router.get("/products")
def get_products():
    return get_all_products_service()

@router.get("/products/{product_id}")
def get_product(product_id: int):
    product = get_product_service(product_id)
    if product:
        return product
    return {"error": "Not found"}

@router.put("/products/{product_id}")
def update_product(product_id: int, product: Product):
    updated = update_product_service(product_id, product)
    if updated:
        return updated
    return {"error": "Not found"}

@router.delete("/products/{product_id}")
def delete_product(product_id: int):
    deleted = delete_product_service(product_id)
    if deleted:
        return {"message": "Deleted"}
    return {"error": "Not found"}
```

---

## ✅ 5. main.py

```python
from fastapi import FastAPI
from routes.product_routes import router as product_router

app = FastAPI()

app.include_router(product_router)
```

| Hard to scale      | Easy to scale        |
| Messy              | Clean                |
