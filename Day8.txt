# 🧠 पहले समझो — Why SQLAlchemy?

तुमने सही बोला 👇

* **Pydantic → API layer (input/output validation)**
* **SQLAlchemy → Database layer (store/retrieve data)**

लेकिन deep समझो 👇

👉 Problem:
अगर हम direct SQL लिखें:

```sql
INSERT INTO products VALUES (...)
```

तो:

* Hard to manage
* Error-prone
* Not Pythonic

👉 Solution: **SQLAlchemy (ORM)**

👉 ORM = Object Relational Mapping

मतलब:

```python
product.name = "Laptop"
```

➡ DB में save हो जाएगा

# 🧱 Step 3: SQLAlchemy Model (DB Table)

📄 `models/product_model.py`

```python
from sqlalchemy import Column, Integer, String
from database.db import Base

class ProductDB(Base):
    __tablename__ = "products"

    id = Column(Integer, primary_key=True, index=True)
    name = Column(String)
    price = Column(Integer)
    qty = Column(Integer)
```

👉 ये actual DB table है

---

# 📦 Step 4: Pydantic Schema

📄 `schemas/product_schema.py`

```python
from pydantic import BaseModel

class ProductCreate(BaseModel):
    name: str
    price: int
    qty: int

class ProductResponse(ProductCreate):
    id: int

    class Config:
        from_attributes = True
```

👉 क्यों?

* Input और Output अलग रखने के लिए
* Security + Clean API

# 🔧 Step 5: Service Layer

📄 `services/product_service.py`

```python
from models.product_model import ProductDB
from database.db import SessionLocal

def create_product_service(product):
    db = SessionLocal()

    new_product = ProductDB(
        name=product.name,
        price=product.price,
        qty=product.qty
    )

    db.add(new_product)
    db.commit()
    db.refresh(new_product)

    db.close()
    return new_product


def get_products_service():
    db = SessionLocal()
    products = db.query(ProductDB).all()
    db.close()
    return products
```

👉 यहाँ actual DB interaction हो रहा है

# 🌐 Step 6: Routes

📄 `routes/product_routes.py`

```python
from fastapi import APIRouter
from schemas.product_schema import ProductCreate, ProductResponse
from services.product_service import *

router = APIRouter()

@router.post("/products", response_model=ProductResponse)
def create_product(product: ProductCreate):
    return create_product_service(product)


@router.get("/products", response_model=list[ProductResponse])
def get_products():
    return get_products_service()
```

# 🚀 Step 7: Main File

📄 `main.py`

```python
from fastapi import FastAPI
from database.db import Base, engine
from routes.product_routes import router

app = FastAPI()

Base.metadata.create_all(bind=engine)

app.include_router(router)
```
